Wat is een web service?

Een webservice is een applicatie die draait op een server en in staat is met andere applicaties op andere computers te communiceren.

Een webservice maakt het mogelijk om op afstand (meestal over het Internet) vanaf een client (een (web)applicatie of component) een dienst op te vragen aan een server, bijvoorbeeld het maken van een berekening, het leveren van gegevens of het uitvoeren van een taak. Webservices spelen een groeiende rol in het denken over component-based systems.

Waar gaat onze webservice draaien?

Webservices kunnen op verschillende machines gehost worden. Aangezien we in dit vak met Microsoft technologie werken, nemen we als voorbeeld een service gebouwd in WCF

Het is mogelijk om een service te hosten via je eigen laptop. Deze moet dan wel altijd aanstaan en aangesloten zijn met het internet. Anders is je service niet altijd bereikbaar.

Daarnaast is het mogelijk om zelf een Windows server aan te schaffen en hier de bijbehorende software op te installeren. Je server heeft dan een IIS omgeving nodig. Hierbinnen is het mogelijk je applicatie te laten draaien (Dit kan bijvoorbeeld ook een PHP applicatie zijn).

Wij gaan gebruik maken van de Azure omgeving. Als student van Avans hogeschool heb je een Azure pass ontvangen waarmee je projecten kan hosten op Azure. Dit is ook mogelijk met een gratis dreamspark account.
