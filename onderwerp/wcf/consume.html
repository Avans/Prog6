Service consumeren

Ook voor het consumeren van onze webservice gaan we gebruik maken van een wizard in Visual studio. Via het mapje References is het mogelijk een service reference toe te voegen. Visual studio zal kijken of er op een bepaalde url een webservice draait, en of hij deze kan gebruiken. De rest gebeurd van zelf!

Als de toevoegen van de service gelukt is kunnen we de service gaan aanroepen. Dit doen we via de klasse die visual studio voor ons heeft aangemaakt.

Het is mogelijk dat je de service helemaal veranded, door bijvoorbeeld methode namen te veranderen. In dit geval moeten we onze service reference helaas ook updaten. Dit kan eenvoudig via een update knop op de service. 

ChatService.IChatService service = new ChatService.ChatServiceClient();
Console.WriteLine("Loading messages...");
String[] messages =  service.GetMessages();
PrintMessages(messages);
