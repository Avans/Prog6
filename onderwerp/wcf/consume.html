Service consumeren

Ook voor het consumeren van onze webservice gaan we gebruik maken van een wizard in Visual studio. Via het mapje Refrences is het mogeleijk een service refrence toe te voegen. Visual studio zal kijkenn of er op een bepaalde url een webservice draait, en of hij deze kan gebruiken. De rest gebeurd vanzelf!

Als de toevoegen van de service gelukt is kunnen we de service gaan aanroepen. Dit doen we via de klasse die visual studio voor ons heeft aangemaakt.

Het is mogelijk dat je de service helemaal veranded, door bijvoorbeeld methode namen te veranderen. In dit geval moeten we onze service refrence helaas ook updaten. Dit kan eenvoudig via een update knop op de service. 

ChatService.IChatService service = new ChatService.ChatServiceClient();
Console.WriteLine("Loading messages...");
String[] messages =  service.GetMessages();
PrintMessages(messages);
