Azure

Het is mogelijk om onze service applicatie te starten via visual studio. Een service moet daarentegen 24/7 beschikbaar zijn voor andere applicaties. Hierdoor is het slimmer om onze WCF service te laten draaien op een server.

Er zijn verschillende omgevingen waar je je applicatie kan hosten. Het is vereist dat op de omgeving waarin je wilt hosten IIS (Internet Information Services) geinstalleerd staat. Dit is een verzameling van serverdiensten gemaakt door Microsoft.

Naast deze fysieke servers, is het ook mogelijk om je applicatie te hosten in de cloud. Microsoft bied je een cloud omgeving aan genaamd Azure. Hier heb je niet alleen de mogelijkheid om applicaties te hosten, maar nog veel meer.

Wij gaan onze applicatie hosten in de cloud.

Als het goed heb je van je vakdocent een Azure pass ontvangen. Deze kun je activeren op de website Microsoft Azurepass. Als je geen pas hebt ontvangen, kan je deze aanvragen bij je vakdocent. Het is ook mogelijk om een volledig gratis account aan te vragen via Dreamspark

Na deze stappen staat de webservice online. Nu is het mogelijk voor andere applicaties om deze service te consumeren.
