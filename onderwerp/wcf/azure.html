Deploy

Er zijn verschillende manieren om een applicatie te deployen op een server. Wij gaan gebruik maken van de ingebouwde functionaliteit in Visual Studio.

Het is namelijk mogelijk om een project vanuit Visual Studio te publishen. Om dit te bereiken moeten we een aantal stappen van een wizard doorlopen.

In de sheets van het werkcollege zijn deze stappen terug te vinden.

Firewall

Windows Azure heeft een firewall waar je toestemming moet verlenen tot je SQL server. Ga naar je SQL database > SQL server > Instellingen > Firewall en voeg hier je locale IP adres aan toe.

Tip:Voeg de range 0.0.0.0 tot 255.255.255.255 toe om iedereen toegang te geven.

Webservice

Na deze stappen staat de webservice online. Nu is het mogelijk voor andere applicaties om deze service te consumeren.

Daarnaast kun je statistieken over je webservice bekijken, en is je webservice schaalbaar. Dit houdt in dat het mogelijk is om de capaciteit van je service te verhogen. Heb je vandaag 10 gebruikers, en morgen 1 miljoen? Dat is geen probleem. Zolang je het budget hebt om je service te upgraden!
