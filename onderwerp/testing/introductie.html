Testing

Introductie

Deze week gaan we ons verder verdiepen in het onderdeel unit testing. Dit is vorig blok ter sprake gekomen, maar zullen we dit blok intensiever moeten gaan toepassen. Laten we beginnen met een korte herhaling van vorig blok.

In Visual Studio kun je een Unit Test Project toevoegen aan je solution, waar je kleine stukjes code kunt schrijven die methodes voor je kunnen aanroepen en de output van deze methodes kunnen 'asserten'.

Dit project bestaat uit classes met een TestClass attribuut, alswel methodes met een TestMethod attribuut. Deze methodes kun je uitvoeren via de Test Window in visual studio. Hier kun je zien of een test geslaagd is en zo niet, wat daar dan de reden van is.

Tip: De test explorer kun je vinden onder Test > Window > Test Explorer

Tip: Bekijk ook eens deze coole infographic! Met dank aan

Example:

 
[TestClass]
public class CalculatorTests
{
    [TestMethod]
    public void Add_Return_Correct_Value()
    {
        // 1. Arrange
        var calculator = new Calculator();
        // 2. Act
        int result = calculator.Add(2, 3, 4);
        // 3. Assert
        Assert.AreEqual(9, result);
    }
}
