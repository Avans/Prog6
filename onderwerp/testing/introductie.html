Testing

Introductie

Deze week gaan we ons veder verdiepen in het onderdeel unit testing. Dit is vorig blok terspraken gekomen, maar zullen we dit blok intensiever moeten gaan toepassen. Laten we beginnen met een korte herhaling van vorig blok.

In Visual Studio kun je een Unit Test Project toevoegen aan je solution, waar je kleine stukjes code kan schrijven die methodes voor je kunnen aanroepen en de output van deze methodes kunnen 'asserten'.

Dit project bestaat uit classes met een TestClass attribuut, alswel methodes met een TestMethod attribuut. Deze methodes kun je uitvoeren via de Test Window in visual studio. Hier kun je zien of een test geslaagd is en zo niet, wat dan de reden is dat de test niet gelukt is.

Tip: De test explorer kun je vinden onder Test > Window > Test Explorer
