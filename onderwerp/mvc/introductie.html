MVC

Introductie

PHP fan boys, leg je Notepad++ weg, stop je Webstorm in de kast. We gaan kijken hoe web development er uit ziet met een Microsoft bril. We gaan aan de slag met ASP.net MVC.

ASP.NET MVC 5.0 is een technologie die gebaseerd is op het MVC design pattern (wie zag die aankomen). Het is bovenop het ASP.NET framework gebouwd met specifieke templates en wizards.

De server kant van MVC bevat altijd modellen en controllers gebaseerd op .NET. De view daarentegen kan worden gegenereerd door verschillende view engines. Wij gaan aan de slag met Razor.

Voordelen ASP.NET MVC:

  • Separation Of Concerns
  • Testable logic (controller)
  • WYSIWYG (view)
  • Sluit goed aan op HTML5/CSS3/jQuery

Model

  • Domainmodel
  • Businesslogic
  • Persisteren van data (DB)
  • Repositories
  • Viewmodel

Controller

  • De spin in het web
  • Access tot het model
  • Bepaalt de view
  • Base class Controller
  • Methods in een controller heten Actions

View

  • Rendered output
  • HTML, CSS, JavaScript
  • Server-side scripting
  • Multiple view engines
    • Bijv Razor.
