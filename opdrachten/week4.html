We geen aan League manager bouwen. Dit is een spel waarbij je je eigen Leauge of legends team kan samenstellen en kan spelen tegen andere spelers.

Om te beginnen moet je het Start project forken. In deze versie kun je met 2 man aan deze opdracht werken.

In de solution LeagueMananger zitten de volgende projecten

  • LeagueManager.Domain
  • LeagueManager.Service
  • LeagueManager.Test
  • LeagueManager.WPF

De flow van het spel

  1. Stel een team samen van champions
  2. Stuur deze samenstelling naar de server
  3. Kijk elke seconde of er een andere speler een samenstelling heeft verstuurd
  4. De server kijkt welke samenstelling gewonnen heeft en stopt deze in een game resultaat
  5. Haal de resultaten op van de server

De opdracht

Bouw de service verder af en sluit deze aan op de client applicatie. Deze service moet een OperationContract hebben waarmee je je gekozen opstelling naar de server kan verrsturen. Daarnaast moet de server een OperationContract hebben waarmee je het resultaat van de 'match' kan ophalen. Indien de 'match' resultaten direct na het versturen van je opstelling beschikbaar zijn, moet deze methode de resultaten meteen terug geven.

De service moet ook een OperationContract hebben die een lijst van alle te spelen champions terug geeft.

In het domein model zit een interface IChampion. Gebruik dit interface om voor minstens 5 champions een klasse te defineren. Gebruik hierna Ninject om een volledige lijst met champions op te halen.
