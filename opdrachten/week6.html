Grand Finale

Workshop assignment ws Workshop Assignment

De laatste workshop opdracht. Deze week geen start project, maar even alles van de grond af opbouwen.

Wat moet je deze week maken? Een hele simpele applicatie. Namelijk een website die een lijst van pokemons kan laten zien, en waar je nieuwe pokemons aan toe kan voegen.

Wat is er dan zo moeilijk aan deze opdracht? Alles moet netjes volgens de regels!

Opdracht 1

Maak een WCF service die een lijst van pokemons uit de database terug kan geven, en die een nieuw pokemon object aan de database kan toevoegen

Opdracht 2

Maak een MVC applicatie die deze WCF service consumeert met een pagina voor alle pokemons, en een pagina om nieuwe pokemons toe te voegen.

Opdracht 3

Voeg validatie toe op de create pagina van de pokemon met de volgende eisen:

  • Alle eigenschappen van een Pokemon zijn verplicht
  • De geboortedatum moet in het verleden liggen.
  • Het Level moet een waarden hebben tussen de 0 en 30.
  • De Naam en het Type is een string tussen de 2 en 50 characters.
