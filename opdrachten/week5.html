League Store

Workshop assignment ws Workshop Assignment

We gaan weer verder bouwen aan onze website http://leaguestore.azurewebsites.net/.

Bouw verder in het project van vorige week of clone het start project als je dit nog niet gedaan hebt. We hebben minstens een Bundle controller die een lijst laat zien van alle bundles. Je kan altijd even in de uitwerking spieken als je er niet uit komt.

De opdracht

Breid de BundelController uit met een Post methode. Deze methode toont een pagina aan de gebruiker waar hij informatie over een bundel kan invoeren.

De gebruiker moet de volgende gegevens kunnen invoeren:

  • Name
  • Description
  • RiotPoints
  • Email adres

Het moet voor de gebruiker ook mogelijk zijn om producten te kiezen waar de bundel uit bestaat. Gebruik hiervoor het html select element met MultiSelect.

Om dit voor elkaar te krijgen moet je nadenken welke informatie er naar de server gestuurd wordt over de gekozen producten. Zijn dit de volledige producten? Of bijvoorbeeld alleen de Id's?

Tip: Om alle producten in de select box te krijgen moet je gebruik maken van een View Model of de ViewBag

In de workshop opdracht hebben we het scherm gemaakt om nieuwe bundels toe te voegen. Nu gaan we bestaande bundels updaten.
