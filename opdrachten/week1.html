To Test or not to Test

Workshop assignment ws Workshop Assignment

Een bekende tovenaar heeft zoveel spreuken, dat hij zelf niet meer weet of hij ze goed uitvoert. Om deze reden huurt hij een software ontwikkelaar in (jij) om dit voor hem te controlleren.

Als bijlage krijg je het toverboek van de tovenaar en met dit boek ga je unit test schrijven om te controlleren of alle spreuken wel goed werken

Hieronder staat een link naar het toverboek van de tovenaar. Daarnaast is er via github een class library beschikbaar met daarin de klasse Wizard.

Schijf per spreuk 2 unit testen om te controlleren of de tovenaar de spreuk goed uitvoert

  • Een Test om te controlleren of de spreuk werkt met de juiste input
  • Een Test om te controlleren wat er gebeurd als er foute input wordt gebruikt
